Git and Github

  • What is the difference between Git and Github?

Git acts as a version control for files that you are working with. It allows user/s to track and manage multiple changes and commit these changes to the main file (branch) being worked on.

Github is a cloud based service to host multiple repositories. It works in conjunction with Git cooridinating to host multiple projects and allow users to acces these projects from anywhere
